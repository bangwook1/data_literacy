# 1️⃣ 개념정리

## 01. 결과와 결론은 다르다

```
✅ 학습 목표 :
* 데이터 분석 결과와 해석을 통해 내리는 결론을 구분되어야 함을 이해한다.
```

### 결과와 결론의 관계  
- 결과에 대한 **설명을 통해 결론을 도출**해야 한다.  
- 반대로, **결론은 결과에 기반해 도출**되어야 한다.  
- 결과와 결론은 **본질적으로 동일하지만 표현 방식이 다르다.**  
- 그래프를 단순히 읽어 설명하는 데 그치지 말고, **상대방에게 자신의 의사를 전달**해야 한다



## 02. '결과'와 '결론'의 차이 : 데이터를 너무 단순하게 이해한 사례 

```
✅ 학습 목표 :
* 데이터 수치를 지나치게 단순하게 해석할 때 발생할 수 있는 오류를 사례를 통해 파악한다. 
* 동일한 결과라도 결론은 다양하게 나올 수 있음을 인식하고, 관점의 중요성을 설명할 수 있다. 
```

### 정보의 가치와 관점  
- **정보의 가치를 고려해야 하며**, 그 정보가 **얼마나 높은 가치를 가지는지**가 중요하다.  
- 같은 정보를 보고도 **다양한 해석이 가능**하므로, **어떤 관점과 주안점**에서 바라보는지가 핵심이다.  
- 단순한 사실 나열보다, **타당한 근거로 상대를 설득하고 설명**할 수 있어야 한다.


## 03. 결론을 낼 때 주의해야 할 점

```
✅ 학습 목표 :
* 결론 도출 시 반드시 점검해야할 부분을 습득한다. 
* 데이터를 통해 얻은 인사이트를 왜곡 없이 요약하는 방법을 설명할 수 있다. 
```

### 분석 태도와 주의점  
- **주관을 일부 반영할 수 있으나**, 과도한 주관 개입은 금물이다.  
- **성급한 일반화**를 피해야 하며,  
- **인지편향(선입견)** 또한 경계해야 한다. 사람은 흔히 자신이 생각한 이유를 정당화하려는 **습성**이 있다.  
- 따라서, **정답은 하나로 고정되지 않는다.** 다양한 해석이 존재할 수 있음을 인지해야 한다.
